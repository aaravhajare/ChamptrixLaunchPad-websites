📖 What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web. It describes the structure of a web page using a series of elements and tags.

HTML was first developed by Tim Berners-Lee in 1991 and has since evolved through multiple versions, with HTML5 being the latest major version that introduced many new semantic elements and APIs.

Every website you visit is built on HTML as its foundation. Combined with CSS for styling and JavaScript for interactivity, HTML forms the backbone of the modern web.

💻 Basic HTML Code Example

Below is a simple example of a basic HTML page structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first HTML page.</p> <a href="#">Click here</a> </body> </html>

This simple structure includes the document type declaration, html root element, head section with metadata, and the body containing visible content.

📋 HTML Element Reference

Here's a quick reference table of important HTML elements and their purposes:

Element Type Description
<head> Metadata Contains page metadata like title, charset, and links
<body> Content Contains all visible page content
<nav> Semantic Defines navigation links section
<section> Semantic Defines a thematic section of content
<article> Semantic Represents self-contained independent content
<footer> Semantic Defines footer for document or section
<form> Interactive Creates user input form
<canvas> Graphics Drawing surface for graphics via JavaScript
